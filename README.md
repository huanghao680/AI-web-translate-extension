# AI 网页翻译

[English](README_en.md)

> **⚠️ 早期开发阶段** — 本项目由 AI 大语言模型辅助编写，可能存在 bug，欢迎提交 Issue 和 PR。

基于大语言模型 API 的 Chrome 浏览器翻译插件。

## 功能特性

- **整页翻译** — 一键翻译整个网页，自动跳过表格、代码块、图标字体
- **划选翻译** — 选中文本后点击按钮确认翻译
- **选择块翻译** — 鼠标选择内容块，方向键调整范围
- **总结性翻译** — 先总结再翻译，降低 Token 消耗
- **选词翻译** — 选中单词弹出翻译浮窗
- **原文/译文切换** — 翻译后一键切换
- **多 API 配置** — 支持 DeepSeek、MiMo 等，弹窗内切换
- **翻译风格** — 默认/正式/简洁/学术四种风格
- **Token 统计** — 显示上次和累计 Token 消耗
- **内容精简（Beta）** — 仅提取正文，减少 Token
- **配置导出/导入** — JSON 文件备份与迁移
- **错误日志** — API 请求失败时记录详情
- **本地存储** — 敏感数据仅存本地
- **暗色模式** — 跟随系统深色主题
- **多语言** — 支持中文/English 界面切换
- **SPA 自适应** — 单页应用自动重置状态

## 快速开始

1. `chrome://extensions/` → 开发者模式 → 加载已解压的扩展
2. 点击扩展图标 → 设置 → 新建配置，填写 Base URL / API Key / 模型
3. 保存配置 → 测试连接

## 配置管理

支持多个 API 配置，每配置独立存储 `baseUrl / apiKey / model / maxTokens`。设置页 CRUD，弹窗中切换。

## 翻译风格

| 风格 | temperature | 说明 |
|------|------------|------|
| 默认 | 0.3 | 通顺自然 |
| 正式 | 0.15 | 书面语 |
| 简洁 | 0.4 | 精炼简略 |
| 学术 | 0.1 | 术语一致 |

## API 兼容

使用 OpenAI 兼容格式。

- **DeepSeek**: `https://api.deepseek.com` / `deepseek-v4-flash` / 最大 384K tokens
- **MiMo**: `https://api.xiaomimimo.com/v1` / `mimo-v2.5-pro` / 最大 131072 tokens

## 注意事项

- 需自行准备 API Key，费用由 API 服务商收取
- 所有配置仅存储在 `chrome.storage.local`
- 导出文件以明文包含 API Key，请妥善保管
- 多配置切换后需重新翻译页面
